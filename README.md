# ant-pro

## Project setup
```
# 安装包依赖
npm install

# 运行开发环境 localhost:8080
npm run dev

# 打包生产环境,目标目录dist
npm run build

# 分析打包详情
npm run build --report

### Compiles and hot-reloads for development
```
